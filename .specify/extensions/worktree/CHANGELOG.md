# Changelog

## 1.0.0 (2026-04-09)

- Initial release
- Add `/speckit.worktree.create` command for spawning isolated feature worktrees
- Add `/speckit.worktree.list` command for worktree status dashboard
- Add `/speckit.worktree.clean` command for safe worktree cleanup
- Worktrees stored in `.worktrees/` directory (auto-gitignored)
- Optional `after_specify` hook for auto-creating worktrees
- Addresses community request in issue #61 (36+ upvotes)
